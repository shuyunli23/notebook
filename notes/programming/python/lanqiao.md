## Python vs C++ 差异

### 优势
1. **无需手动内存管理**: 不需要 `new`/`delete`
2. **内置高精度**: `int` 自动支持大整数
3. **简洁语法**: 列表推导式、内置函数
4. **动态类型**: 无需声明类型

### 注意事项
1. **性能**: Python比C++慢,大数据量可能超时
2. **输入输出**: 使用 `input()` 和 `print()` 替代 `cin`/`cout`
3. **数组下标**: Python从0开始,需注意索引调整
4. **除法**: Python3中 `/` 是浮点除法,`//` 是整除

## 学习建议

1. **动态规划三部曲**
   - 定义状态 (dp数组含义)
   - 写出状态转移方程
   - 确定初始状态和计算顺序

2. **贪心算法关键**
   - 证明局部最优 → 全局最优
   - 排序往往是关键步骤

3. **图算法要点**
   - 选择合适的数据结构(邻接表/矩阵)
   - BFS用队列,DFS用栈/递归
   - 记录访问状态避免重复

4. **位运算技巧**
   - XOR性质: `a ^ a = 0`, `a ^ 0 = a`
   - 左移相当于乘2,右移相当于除2

## 时间复杂度对比

| 算法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 迷宫DP | O(n×m) | O(n×m) |
| 铠甲合体 | O(n log n + m×n) | O(n) |
| 连连看 | O(n×m×k²) k为相同元素数 | O(n×m) |
| 星际旅行BFS | O(Q×(V+E)) | O(V) |
| 01背包 | O(N×W) | O(N×W) |
| 完全背包 | O(N×W) | O(W) |
| Hanoi递推 | O(n) | O(1) |

## 常见错误

1. **数组越界**: Python会抛出 `IndexError`
2. **整数溢出**: Python自动处理,但C++需要用`long long`
3. **浮点精度**: 使用 `round()` 或格式化输出
4. **输入格式**: 注意空格分隔 vs 换行分隔
